Algebra(ish) Games

What's this?

These are games are designed to (hopefully) help students learn and understand basic concepts in (abstract) algebra. The games utilizes the general definition algebra, which is a set with defined operations that satisfies some constraints.

Status

The games themselves are barely done, but the architecture and framework itself is more or less stable so it shouldn't be too hard to add new grid-based games.
